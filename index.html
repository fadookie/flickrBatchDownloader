This page is a tool for downloading original-size images from Flickr photosets. Given a photoset ID, it will generate a page of links to the original size photos, if available. You can then use a download manager such as DownThemAll! for Firefox to save the photos. You could also copy-paste the links into a batch download tool such as DownloadAll for Chrome, wget or curl. You may change the delimiter string to format output for copy-pasting.

This tool makes use of Yahoo Query Language and is written in pure JavaScript.
